## ReactNativeAppAlbums
* git clone https://github.com/padmacnu/ReactNativeAppAlbums.git
* npm install

## Running
* react-native run-ios


## Setting up React Native for the first time on Mac
* /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
* brew update // if you already have brew
* brew install node
* node -v // confirm node is installed and check the version
* brew install watchman // watchman by facebook, look for changes in hard rive
* npm install react-native-cli

## Creating React Native App
* react-native init albums
* react-native run-ios 
* or
* react-native start --port=8088
